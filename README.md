# ByteCore - Responsive Web Page

This project is a responsive web page developed with HTML and CSS. The goal is to present a company website, with sections for services, about, team, contact, and location. The design is modern and adapted for different devices.

## Overview

The page includes:

- **Navigation menu** for desktop and mobile
- **Hero section** with slogan and description
- **Services presentation** offered by the company
- **About the team** highlighting a founding member
- **Contact form** for visitors to send messages
- **Interactive map** to facilitate the company's physical location
- **Footer** with contact information

---

## How to Use

1. **Clone or download the project:**

    ```bash
    git clone https://github.com/lu1zviana/ByteCore
    ```

2. **File structure:**

    - `index.html` (main page)
    - `css/style.css` (styles)
    - `images/` (site images, e.g., mockup.jpg)

3. **Open the `index.html` file in your browser.**

    No server or extra configuration is required; just open the file locally.

---

## Project Structure

```text
/
├── index.html
├── css/
│   └── style.css
├── images/
│   └── mockup.jpg
```

- The `index.html` file contains the entire HTML structure and links to the CSS.
- CSS is in `css/style.css` for styling and responsiveness.
- All images used should be saved in the `images/` folder.

---

## Customization

- **Content:** Edit text, titles, and images directly in the HTML.
- **Styles:** Modify `css/style.css` to change colors, fonts, or layout.
- **Menu:** Add or remove items as needed.
- **Form:** You can integrate the contact form with backend services to receive messages by email.

---

## Features

- **Responsive:** The site adapts to mobile devices and desktops.
- **External fonts:** Uses Google Fonts for a modern look.
- **Integrated map:** Displays the physical address using an embedded Google Map.
- **Team:** Presents members with photos and descriptions.
- **Contact:** Form for visitor interaction.
